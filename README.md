## Build

### 安装环境

本项目需要安装 `Python` 和 `Nodejs`，执行 `pip install -r requirements.txt` 和 `npm install`。

### Format

执行 `npm format:prettier` 会 Format 全部文件，在 `git commit` 之前也会自动执行这条命令。
